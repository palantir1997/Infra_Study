## 주요 개념 정리

### 온프레미스 (On-premise)
기업이 서버, 네트워크, 스토리지 등 IT 인프라를 자체 데이터 센터에 직접 설치하여 운영하는 방식.
보안성이 높고 완전한 통제권을 가지지만, 초기 구축 비용(CapEx)이 크고 유지보수를 내부에서 담당해야 한다.

### 클라우드 (Cloud Computing)
인터넷을 통해 컴퓨팅 자원을 필요할 때 빌려 쓰는 서비스 방식.
초기 비용 부담이 적고 자원 확장이 유연하며, 사용한 만큼 비용을 지불하는 종량제 모델이 일반적이다.

### ISMS-P
기업의 정보자산 보호 체계가 국가 인증 기준에 적합한지 심사하는 제도.
관리체계 수립 및 운영 / 보호대책 요구사항 / 개인정보 처리 단계별 요구사항의 3개 영역으로 구성된다.

## OSI 7계층 & 네트워크 장비

| 계층 | 이름 | 주요 장비/프로토콜 |
|------|------|-----------------|
| L7 | Application | HTTP, FTP, DNS, SMTP |
| L6 | Presentation | - |
| L5 | Session | - |
| L4 | Transport | TCP, UDP |
| L3 | Network | IP, 라우터 |
| L2 | Data Link | MAC, L2 스위치 |
| L1 | Physical | 케이블, 허브 |

### 캡슐화 / 역캡슐화
- 캡슐화: 송신측에서 상위 계층 → 하위 계층으로 헤더를 붙이며 포장
  - 데이터 → 세그먼트(L4) → 패킷(L3) → 프레임(L2) → 비트(L1)
- 역캡슐화: 수신측에서 하위 계층 → 상위 계층으로 헤더를 제거하며 데이터 추출

## 라우팅 프로토콜 비교

| 프로토콜 | 방식 | 특징 |
|---------|------|------|
| RIP | 거리 벡터 | 소규모, 수렴 느림 |
| EIGRP | 거리 벡터(향상) | 대규모, 빠른 수렴 |
| OSPF | 링크 상태 | 대규모, 빠른 수렴 |

### 정적 라우팅 (Static Routing)
```bash
ip route [목적지 네트워크] [서브넷 마스크] [다음 홉 주소]
```
- 장점: CPU 부하 없음, 보안성 높음
- 단점: 네트워크 변화에 자동 대응 불가, 규모 커지면 관리 복잡

## Cisco Packet Tracer 기본 명령어

### 장비 모드

| 모드 | 프롬프트 | 진입 방법 |
|------|---------|---------|
| User EXEC | Router> | 접속 직후 |
| Privileged EXEC | Router# | enable |
| Global Config | Router(config)# | conf t |
| Interface Config | Router(config-if)# | interface [인터페이스] |

### 자주 쓰는 명령어

```bash
enable                          # 관리자 모드 진입
configure terminal              # 설정 모드 진입
hostname [이름]                  # 장비 이름 설정
enable secret [비번]             # 암호화된 관리자 비밀번호 설정
service password-encryption     # 모든 패스워드 암호화
banner motd                     # 부팅 안내 메시지 설정

interface gig0/0                # 인터페이스 진입
ip address [IP] [마스크]         # IP 주소 설정
no shutdown                     # 인터페이스 활성화

line console 0                  # 콘솔 접속 설정
line vty 0 4                    # 원격 접속 설정 (최대 5명)

show running-config             # 현재 실행 설정 확인
copy running-config startup-config  # 설정 저장
write memory                    # 설정 저장 (단축)

Ctrl + Shift + 6                # 세션 종료
Ctrl + Z                        # 설정 모드 탈출
```
