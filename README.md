## 핵심 규칙

- **PC/Server** → IP + 서브넷 + 게이트웨이 3개 입력
- **라우터 LAN (Gig)** → 해당 대역의 `.1` 번 사용 (게이트웨이 역할)
- **라우터 WAN (Serial)** → 연결되는 두 라우터끼리 같은 대역에서 `.1`, `.2` 나눠가짐
- **게이트웨이** → PC/Server에만 입력, 라우터에는 없음
