## 핵심 개념 정리

주요 개념은 각 실습 노트 안에 정리되어 있습니다.  
공통으로 자주 등장하는 명령어는 아래에 누적 정리 예정.

```
show ip route          라우팅 테이블 확인
show running-config    현재 설정 전체 보기
show ip protocol       동작 중인 라우팅 프로토콜
write memory           설정 저장
```
